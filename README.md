# 💻 BarakaPOS Web Dashboard

BarakaPOS tizimining admin va savdo paneli. Ushbu veb-ilova orqali do'kon egalari savdolarni kuzatishi, mahsulotlar qoldig'ini boshqarishi va hisobotlarni tahlil qilishi mumkin.

## 🚀 Texnologiyalar
* **Framework:** React.js (Vite)
* **Styling:** Tailwind CSS (yoki siz ishlatgan kutubxona)
* **State Management:** React Context API / Redux
* **API Client:** Axios
